# Template: Articulo SEO para Blog

## Instrucciones
Genera un articulo SEO completo de 800-1500 palabras optimizado para posicionamiento
en buscadores, con meta tags, estructura de headings e internal linking.

## Contexto del Negocio
- Negocio de venta de articulos novedosos importados de China
- Mercado: LATAM (Mexico, Colombia, Argentina, Chile, Peru)
- Productos: gadgets, accesorios tech, hogar, belleza, cocina, juguetes

## 1. META TAGS

### Meta Title (maximo 60 caracteres)
- Incluir keyword principal al inicio
- Formula: "{Keyword} | {Beneficio/Anio} - {Marca}"
- Ejemplo: "Gadgets Baratos de Aliexpress | Guia Completa 2026"

### Meta Description (maximo 160 caracteres)
- Incluir keyword principal
- Incluir un beneficio claro
- CTA implicito (Descubre, Conoce, Aprende)
- Ejemplo: "Descubre los mejores gadgets baratos importados de China. Guia actualizada con precios, links y reviews honestos para comprar inteligente."

## 2. ESTRUCTURA DEL ARTICULO

### H1: Titulo Principal
- Incluir keyword exacta
- Natural y atractivo para el lector
- Solo 1 H1 por articulo

### Introduccion (150-200 palabras)
- Parrafo 1: Hook - Problema o necesidad del lector
- Parrafo 2: Contexto - Por que es relevante ahora
- Parrafo 3: Promesa - Que va a encontrar en este articulo
- Incluir keyword principal en los primeros 100 caracteres

### H2: Secciones Principales (3-6 secciones)
- Cada H2 debe ser descriptivo y contener keyword o variacion
- Debajo de cada H2: 100-250 palabras
- Usar listas (ul/ol) cuando sea posible
- Incluir datos, numeros, precios concretos

### H3: Subsecciones (0-3 por cada H2)
- Para desglosar temas complejos
- Ideal para "Pros y Contras", "Caracteristicas", etc.

### Conclusion (100-150 palabras)
- Resumen de puntos clave
- Recomendacion personal
- CTA: "Visita nuestra tienda", "Comparte este articulo"
- Pregunta para generar comentarios

## 3. OPTIMIZACION SEO

### Densidad de Keywords
- Keyword principal: 1-1.5% del texto total
- Keywords secundarias: mencionar 2-3 veces cada una
- Keywords LSI (semanticamente relacionadas): incluir naturalmente

### Estructura Tecnica
- Parrafos cortos: 2-4 oraciones maximo
- Oraciones cortas: 15-20 palabras promedio
- Voz activa predominante
- Transiciones entre secciones
- Listas con viñetas para escaneo rapido

### Internal Linking (2-4 links)
- Sugerir anchor texts naturales
- Indicar slugs/URLs sugeridos para otros articulos del blog
- Ejemplo: "Si te interesa, tambien puedes ver nuestra [guia de productos virales](/productos-virales-tiktok)"

### Imagenes Sugeridas (3 minimo)
- Indicar donde colocar imagenes en el articulo
- Describir que deberia mostrar cada imagen
- Incluir sugerencia de alt text con keyword

```markdown
[...mas secciones...]

## Conclusion

{Resumen + CTA + pregunta}
```

## Reglas Generales
- Todo en ESPANOL formal pero accesible
- 800-1500 palabras totales
- Keyword density: ~1.5%
- Tono: profesional, informativo, util
- Evitar relleno - cada oracion debe aportar valor
- No usar cliches SEO ("en este articulo vamos a...")
- Datos y precios concretos cuando sea posible
- El articulo debe ser util incluso sin comprar nada
